Use this Skill for general questions about items, blocks, recipes, mechanics, or progression.

1. Resolve ambiguous names before making factual claims.
2. Search indexed knowledge and fetch the full relevant documents.
3. Use live recipe or player-context tools for facts that may differ by pack or server.
4. State which source supports each important instruction. Never invent an unavailable recipe or guide entry.
5. If evidence is unavailable or contradictory, say exactly what could not be verified and suggest the next observable check.

Answer in the player's language. Prefer concise ordered steps for procedures.
