# OpenAllay 0.1.1 Emergency Context Fix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Prevent unbounded Tool results from exhausting model/UI context, make exact results progressively retrievable, repair cross-provider metadata refresh/cache, add fixed-origin MC百科 article reading, and publish `0.1.1-SNAPSHOT` with retained verification.

**Architecture:** Normalize each Tool result exactly once, then pass it through a common governor that either forwards it or retains the exact JSON behind an owner-bound opaque handle and emits a bounded projection. Re-enter context preparation before every model continuation, render only bounded asynchronous detail pages, resolve canonical model metadata independently of the inference host, and route MC百科 article bodies through the same progressive resource layer.

**Tech Stack:** Java 25, Gson, JDK virtual threads/HTTP client, Minecraft 26.2 common UI projections, JUnit 5, Gradle, SQLite history, GitHub Actions, Modrinth v2 API.

```java
public final class ToolResultGovernor {
    public static final int MODEL_RESULT_BYTES = 8 * 1024;
    public ToolResultProjection project(ToolResultOwner owner, JsonObject exact) {
        byte[] bytes = canonicalBytes(exact);
        if (bytes.length <= MODEL_RESULT_BYTES) return ToolResultProjection.inline(exact);
        String ref = store.retain(owner, exact);
        JsonObject bounded = projector.project(exact, store.describe(owner, ref).orElseThrow());
        if (canonicalBytes(bounded).length > MODEL_RESULT_BYTES) {
            bounded = projector.minimal(exact, store.describe(owner, ref).orElseThrow());
        }
        return ToolResultProjection.spilled(bounded, ref);
    }
}
```

The projector adds only whole JSON values. It preserves normalized status,
failure code/message, evidence, stable references, exact counts, schema paths,
and the resource descriptor. Assert the minimal form always fits.

```java
public sealed interface Match permits Exact, UniqueAlias, Missing, Ambiguous {}

static Match match(String configuredId, Collection<ModelMetadata> trusted) {
    List<ModelMetadata> exact = trusted.stream()
            .filter(it -> it.providerModelId().equals(configuredId)
                    || it.canonicalModelId().equals(configuredId)).toList();
    if (exact.size() == 1) return new Exact(exact.getFirst());
    String leaf = configuredId.substring(configuredId.lastIndexOf('/') + 1);
    List<ModelMetadata> aliases = trusted.stream()
            .filter(it -> it.canonicalModelId().endsWith("/" + leaf)).toList();
    return aliases.size() == 1 ? new UniqueAlias(aliases.getFirst())
            : aliases.isEmpty() ? new Missing() : new Ambiguous(aliases.size());
}
```

Fetch trusted metadata independently of profile host. Cache canonical identity
and alias match. Return per-profile `UPDATED`, `CACHE_HIT`, `UNCHANGED`,
`NOT_FOUND`, `AMBIGUOUS`, or `SOURCE_UNAVAILABLE`; aggregate UI text lists the
actual counts.
